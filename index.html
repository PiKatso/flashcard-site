JavaScript Flashcards

JavaScript and jQuery practice!

Use these flash cards to practice terms and learn concepts from both JavaScript and jQuery.

JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Variables

JavaScript variables are containers for storing data values.

Variable Naming Conventions

lowerCamelCase

Functions

JavaScript functions are defined with the function keyword. A block of code that performs an action and returns a result.

Methods

An action run on a piece of data.

Arguments

Arguments provide the method with additional information.

Parameters

A variable that is assigned to the argument.

Return

The method or functions response/answer.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

Exact text enclosed in quotes.

Booleans

True or false. They are not strings, they simple represent being true or false.

Undefined

Not defined

NaN

Not a number, a type of javascript number.

Escape

The escape() function encodes a string. This function encodes special characters, with the exception of: * @ - _ + . /

Alert

A function that opens a dialogue box and returns undefined

Comments

Code that is not read by JavaScript

jQuery

A JavaScript library.

Attributes

Additional information provided through and HTML tag.

DOM

Document Object Model
